ARTWALK ASSETS FOLDER
=====================
Place artwork photos here. Supported formats: .jpg / .jpeg / .png / .webp

To assign a photo to an artwork, update the `userPhoto` field in artwalk_2.html:

  userPhoto: "artwalk_assets/your_photo.jpg"

Keep artwalk_2.html and the artwalk_assets/ folder in the same directory
so the relative paths resolve correctly.
